\doxysubsection{Detailed Description}
Represents a 4D vector with four float components. 

The {\ttfamily \doxylink{structshguivec4}{shguivec4}} structure is used to represent 4D vectors with {\ttfamily x}, {\ttfamily y}, {\ttfamily z}, and {\ttfamily w} components. It is commonly employed in {\ttfamily shgui} applications for various 4D operations.
