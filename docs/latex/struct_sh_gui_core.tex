\doxysubsection{Detailed Description}
Represents the core Vulkan components for the {\ttfamily shgui} library. 

The \doxylink{struct_sh_gui_core}{Sh\+Gui\+Core} structure encapsulates essential Vulkan components required by the {\ttfamily shgui} library for rendering and GUI operations. It includes the Vulkan device, physical device, graphics queue information, rendering surface, render pass, and other relevant settings.
