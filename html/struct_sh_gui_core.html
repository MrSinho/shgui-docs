Detailed Description

Represents the core Vulkan components for the shgui library.

The ShGuiCore structure encapsulates essential Vulkan components required by the shgui library for rendering and GUI operations. It includes the Vulkan device, physical device, graphics queue information, rendering surface, render pass, and other relevant settings.
