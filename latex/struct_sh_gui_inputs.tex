\doxysubsection{Detailed Description}
Represents {\ttfamily shgui} input data. 

The {\ttfamily \doxylink{struct_sh_gui_inputs}{Sh\+Gui\+Inputs}} structure encapsulates various input data used in {\ttfamily shgui} applications, including window dimensions, key events, and a system-\/submitted character. It is designed to help manage and process input-\/related information.
